Sobre mim

Atualmente, tenho focado meus estudos em umas das principais linguagens para desenvolvimento web: JavaScript, tanto para frontend quanto backend com Node.js. Nesse local você poderá encontrar tanto meus projetos pessoais, quanto os que desenvolvo em treinamento para aprender a tecnologia.


No 2º semestre de 2024 sigo para o 8º período da faculdade de Sistemas de Informação e nesse ambiente pude aprender sobre: Algoritmos e lógica de programação, Estrutura de dados, Programação Orientada a Objetos, Banco de Dados Oracle e MySQL, Noções de UI e UX, Desenvolvimento Mobile e o básico das linguagens: C, C++, C# e PHP.


Entre as minhas soft skills as que mais se destacam são: trabalho em equipe, responsabilidade, adaptabilidade, organização, proatividade e criatividade.
